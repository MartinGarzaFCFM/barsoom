# Saludar JavaScript action

Esta accion imprime "Hola Mundo" u "Hola" + el nombre de la persona a quien saludara.

## Inputs

### `nombre`

**Required** El nombre de la persona a quien saludar. Por defecto `"Mundo"`.

## Outputs

### `saludo`

El saludo en si

## Ejemplo de uso

```yaml
uses: actions/barsoom@e76147da8e5c81eaf017dede5645551d4b94427b
with:
  nombre: Martin Garza
```
